MeteoApp 

## Description :
MeteoApp est une application web météo développée avec React et Vite.
Elle permet d’afficher les informations météorologiques (conditions actuelles, prévisions, etc.) directement dans le navigateur à partir de données fournies par une API météo.

Le projet est entièrement front-end et pensé pour être rapide, moderne et responsive.
Il est également déployé en ligne, ce qui permet de le tester sans installation locale.

## Live Demo

Application en ligne : https://meteoapp.vercel.app

## Fonctionnalités

Recherche météo par ville

Affichage des conditions météorologiques actuelles

Interface simple et responsive

Chargement rapide grâce à Vite

Architecture claire basée sur React

## Technologies utilisées

React — bibliothèque front-end

Vite — outil de build et de développement

JavaScript (ES6+)

HTML5 / CSS3

API météo externe (ex : OpenWeather, WeatherAPI, etc.)
